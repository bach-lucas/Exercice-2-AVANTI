## Visão Técnica

### Stack Principal
- **Front-end:**
  - HTML5 Semântico
  - CSS3 com Pré-processadores (Variáveis CSS nativas)
  - JavaScript ES6+
  - Bootstrap 5.3.0
  - Swiper.js para carrosséis

- **Padrões de Qualidade:**
  - WCAG 2.1 para acessibilidade
  - SEO On-Page otimizado
  - Performance Core Web Vitals

### Dependências
- Font Awesome 6.4.0
- Bootstrap Icons
- Google Fonts
